## Usage

```vim
:ParticleOn       " Syntax color particles
:ParticleRainbow  " Pastel rainbow particles
:ParticleUnko     " Poops particles
:ParticleOff      " Stop
```

### Options

```vim
let g:particle_count = 3  " Number of particles per keystroke (1-10, default: 3)
```

## Requirements

Windows!

## Installation

```
mingw32-make
```
